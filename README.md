# بَرّي (Barri)
تطبيق رحلات برية بين الأردن والدول القريبة (سائق/راكب).

## الأهداف
- MVP: تسجيل/دخول، أدوار، إنشاء/بحث/حجز رحلات، حالات حجز.

## التقنية
- Mobile: Flutter
- API: NestJS + Prisma
- DB: PostgreSQL

## التشغيل (لاحقًا على الماك)
- infra: Docker Postgres
- api: Nest + Prisma
- mobile: Flutter
